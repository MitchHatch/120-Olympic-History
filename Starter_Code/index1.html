120 Years of Olympic History Data analysis

Project Description/Outline:

This dataset contains historical information on the modern Olympic Games, covering all the Games from Athens 1896 to Rio 2016. The dataset consists of 271,000 records, with each row representing an individual athlete participating in a specific Olympic event. The data was sourced from Kaggle. The significance of this dataset is heightened as the USA is gearing up to host the next Olympic Games in 2028.
